# what's this?

KWBRのサンドボックス

どんなことができるのかを参考にしていただくためのリポジトリです。

とはいえ、自分ができることを全部表現しているわけじゃないので参考程度にお願いします。

変なものはprivateにして隠しているので仲良くなって見せても萎えないで欲しいです。。。

# 内容

各ディレクトリ、もしくはファンクションごとにコメントをつけておきます。

作った動機や、その時の気分も書いておくかもしれません。

どこぞの日記感覚でお楽しみください。
